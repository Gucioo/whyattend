Help

This web application is used to track clan war attendance. Clan leaders can upload replays which are parsed for the list of players. Clan members that didn't get to play can sign themselves up as reserve players. This information can be used to track the activity of players and calculate the share of gold for each player.

If you are in one of the clans listed in the players menu, you can log in right away with your Wargaming.net Account. Wargaming does not share any information other than your ingame name with this website, which is used to validate that you are a member of one of the clans configured on this website.

Technical Details

For the technically interested :-) This web application is written in the Python web framework Flask. The database mapping is done with SQLAlchemy. Player authentication is done using Wargaming's capabilities as OpenID provider. The web service API of World of Tanks is used to retrieve further player information such as clan status.

The frontend is written in HTML5, jQuery and Bootstrap.

For further information you can contact the developer.

The plan is to eventually open-source the application to get more developers involved in implementing cool features such as:

  • Track the tank locking status of the clan
  • More statistics about battles and players
  • Integration of more WoT API features and Global Map information
  • Holiday calendar
